Posting on Railway / cloud

No browser window will open on your PC. FastPost runs Chromium headless on the server. That is normal: there is no screen attached in the cloud.

Use Post now and wait for the success or error toast. Confirm the post on Facebook. If login or checkpoints fail, you may need a valid session in PROFILES_DIR (e.g. establish login from a local run once with a visible browser, then persist that profile on a volume). See the README deploy section.
